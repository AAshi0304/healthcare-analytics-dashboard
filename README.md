# Healthcare Analytics Dashboard: Diabetic Patient Readmission Analysis

## Project Overview

This Power BI dashboard analyzes diabetic patient readmissions and helps identify high-risk patients using healthcare data.

## Objective

Identify high-risk diabetic patients and analyze factors influencing hospital readmission.

## Tools Used

* Power BI
* Power Query
* DAX
* CSV Dataset
* GitHub

## Dashboard Features

* Total Patients KPI
* High Risk Patients KPI
* Readmission Rate KPI
* Age Distribution Analysis
* Gender Distribution Analysis
* Race Distribution Analysis
* Number of Diagnoses Analysis
* Hospital Stay Duration Analysis
* Interactive Filters

## Key Insights

* Readmission rate is approximately 47%.
* High-risk patients represent a significant portion of total patients.
* Older age groups have higher admission volumes.
* Diagnosis count shows a relationship with readmission risk.
